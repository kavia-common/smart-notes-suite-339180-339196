# Smart Notes Suite â€” Database Schema (PostgreSQL)

This document describes the database schema created in the `notes_database` container.

Connection command source of truth:
- `notes_database/db_connection.txt` (contains a `psql postgresql://...` command)

## Extensions

The schema uses UUID primary keys generated by `gen_random_uuid()`, provided by `pgcrypto`.

Enabled:
- `CREATE EXTENSION IF NOT EXISTS pgcrypto;`

## Tables

### `users`
Purpose: Optional user accounts for cloud sync.

Columns:
- `id uuid PRIMARY KEY DEFAULT gen_random_uuid()`
- `email text UNIQUE` (nullable to allow anonymous/local-first usage if desired)
- `display_name text`
- `password_hash text` (backend-managed; do not store plaintext)
- `created_at timestamptz NOT NULL DEFAULT now()`
- `updated_at timestamptz NOT NULL DEFAULT now()`

### `notes`
Purpose: Notes content and per-note flags.

Columns:
- `id uuid PRIMARY KEY DEFAULT gen_random_uuid()`
- `user_id uuid REFERENCES users(id) ON DELETE SET NULL`
- `title text NOT NULL DEFAULT ''`
- `content text NOT NULL DEFAULT ''`
- `content_format text NOT NULL DEFAULT 'markdown'`
- `is_pinned boolean NOT NULL DEFAULT false`
- `is_favorite boolean NOT NULL DEFAULT false`
- `archived_at timestamptz` (nullable)
- `deleted_at timestamptz` (nullable; soft delete)
- `created_at timestamptz NOT NULL DEFAULT now()`
- `updated_at timestamptz NOT NULL DEFAULT now()`

Constraints:
- `notes_content_format_chk`: `content_format IN ('markdown','plain','html')`

Indexes:
- `idx_notes_user_updated_at` on `(user_id, updated_at DESC)`
- `idx_notes_deleted_at` on `(deleted_at)`

### `tags`
Purpose: User-scoped tags.

Columns:
- `id uuid PRIMARY KEY DEFAULT gen_random_uuid()`
- `user_id uuid REFERENCES users(id) ON DELETE CASCADE`
- `name text NOT NULL`
- `color text` (optional)
- `created_at timestamptz NOT NULL DEFAULT now()`

Constraints:
- Unique tag name per user: `UNIQUE(user_id, name)`

Indexes:
- `idx_tags_user_name` on `(user_id, name)`

### `note_tags`
Purpose: Many-to-many join between notes and tags.

Columns:
- `note_id uuid NOT NULL REFERENCES notes(id) ON DELETE CASCADE`
- `tag_id uuid NOT NULL REFERENCES tags(id) ON DELETE CASCADE`
- `created_at timestamptz NOT NULL DEFAULT now()`

Constraints:
- Composite primary key: `PRIMARY KEY(note_id, tag_id)`

Indexes:
- `idx_note_tags_tag_id` on `(tag_id)` for reverse lookup (tag â†’ notes)

### `sync_state`
Purpose: Per-user sync metadata (tokens/timestamps/device id).

Columns:
- `user_id uuid PRIMARY KEY REFERENCES users(id) ON DELETE CASCADE`
- `last_sync_at timestamptz`
- `last_sync_token text`
- `device_id text`
- `created_at timestamptz NOT NULL DEFAULT now()`
- `updated_at timestamptz NOT NULL DEFAULT now()`

## updated_at maintenance

A trigger function is created and attached to:
- `users`
- `notes`
- `sync_state`

Function:
- `set_updated_at()` sets `NEW.updated_at = now()` before updates.

## How schema was applied (one statement per psql command)

Each statement was executed individually via:

- `CONN=$(cat db_connection.txt | tr -d '\n')`
- `$CONN -c "SQL_STATEMENT_HERE"`

Statements applied (in order):

1. `CREATE EXTENSION IF NOT EXISTS pgcrypto`
2. `CREATE TABLE IF NOT EXISTS users (...)`
3. `CREATE TABLE IF NOT EXISTS notes (...)`
4. `CREATE TABLE IF NOT EXISTS tags (...)`
5. `CREATE TABLE IF NOT EXISTS note_tags (...)`
6. `CREATE TABLE IF NOT EXISTS sync_state (...)`
7. `CREATE INDEX IF NOT EXISTS idx_notes_user_updated_at ON notes(user_id, updated_at DESC)`
8. `CREATE INDEX IF NOT EXISTS idx_notes_deleted_at ON notes(deleted_at)`
9. `CREATE INDEX IF NOT EXISTS idx_tags_user_name ON tags(user_id, name)`
10. `CREATE INDEX IF NOT EXISTS idx_note_tags_tag_id ON note_tags(tag_id)`
11. `ALTER TABLE notes ADD CONSTRAINT notes_content_format_chk CHECK (content_format IN ('markdown','plain','html'))`
12. `CREATE OR REPLACE FUNCTION set_updated_at() RETURNS trigger ...`
13. `DROP TRIGGER IF EXISTS trg_users_updated_at ON users`
14. `CREATE TRIGGER trg_users_updated_at ... ON users ...`
15. `DROP TRIGGER IF EXISTS trg_notes_updated_at ON notes`
16. `CREATE TRIGGER trg_notes_updated_at ... ON notes ...`
17. `DROP TRIGGER IF EXISTS trg_sync_state_updated_at ON sync_state`
18. `CREATE TRIGGER trg_sync_state_updated_at ... ON sync_state ...`
